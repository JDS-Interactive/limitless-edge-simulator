# Limitless Edge Simulator

**Limitless Edge Simulator** is a browser-based Progressive Web Application (PWA) designed to train users in stock and options trading through realistic, scenario-driven market simulations.

This application provides a risk-free environment where users can learn trading strategies, understand market behavior, and develop disciplined execution habits without using real capital.

---

## 🚀 Features

### 📈 Market Simulation
- Scenario-based market environments:
  - Bull Run
  - Bear Slide
  - Sideways Chop
  - Volatile Breakout
  - Mean Reversion
- Real-time candlestick chart rendering
- Replay mode for historical analysis

---

### 🤖 Strategy Engine
- Built-in trading strategies:
  - RSI Reversal
  - Trend Following
  - Momentum (MACD)
  - Breakout (20-bar)
- Manual trading mode
- Automated strategy execution (simulated)

---

### ⚠️ Risk Management
- Stop Loss (%)
- Take Profit (%)
- Automatic position exits
- Performance tracking

---

### 🎮 Gamification Layer
- XP and Level progression
- Achievements system
- Challenges and Objectives
- Event feed tracking user activity

---

### 🧠 AI Coach (Phase 1.3)
- Context-aware feedback based on:
  - Indicators
  - Strategy output
  - Position status
  - Performance
- Designed for future WebLLM integration (Phase 1.4)

---

### 👤 Profile System
- Multiple local profiles
- Snapshot save/load
- Persistent session state via localStorage

---

### 📱 PWA Support
- Installable on mobile and desktop
- Offline functionality via Service Worker
- Custom app icons and splash behavior

---

## 🧱 Architecture

- Frontend-only (Phase 1.3)
- Vanilla JavaScript (modular structure)
- Canvas-based chart rendering
- Local state persistence (no backend)

---
